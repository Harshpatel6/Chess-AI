Running Chess AI
================

Run the “ChessAI.exe” file. Two windows will open (Console Window, ChessAI GUI Window).

The console window will display the currently installed OpenGL version. 
You must have at least OpenGL Ver.  3.0 or higher to run the GUI.

If there are any installation problems contact “chia8950@mylaurier.ca”

Controls
========

Chess pieces can be selected and moved with the mouse button.
The right hand info panel can be opened and closed where the keys:

	‘O’ = Open’s the side panel.
	‘P’ = Close’s the Side panel.


Information about the project is included in "Chess AI.pdf"
